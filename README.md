# bird-scholar
Repository for WGST-225 semester project.
## Research Question
Do scientific papers describing same-sex behavior in birds cite fewer prior sources than papers describing heterosexual or reproductive behavior?

## MVP
A boiler plate website with some visual data representations.

### For example:

#### Same-sex vs. Heterosexual Behavior Network Density
A network graph with nodes each being a scientific paper, differentiation between same-sex/hetereonormative papers. Scientific papers often cross-reference each other.

Expecting there to be less edges for same-sex behavior papers, making their nodes more isolated.

#### Trajectory of Citations
How fast papers accumulate citations, compare the two different behavior topics
